# Personal Portfolio

视觉设计师 / AI 设计师 / 品牌设计师个人作品集网站。

## Tech Stack

- React
- TypeScript
- Vite
- Framer Motion
- Lucide React
- CSS

## Local Preview

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

构建完成后，发布 `dist/` 目录即可。

## Project Notes

- 页面内容位于 `src/App.tsx`。
- 全局视觉样式位于 `src/index.css`。
- 当前版本按照 Jack 3D Creator landing page 的结构与动效规格还原，并融合了视觉设计师 / AI 设计师 / 品牌设计师定位。
- 网站展示用图片位于 `src/assets/`，已使用压缩后的 display 版本。
- 根目录中的原始作品图可作为后续高清切片、详情页或下载资源使用。
