## 3. 当前工程基线

- React 19 + TypeScript + Vite。
- 主要依赖：Framer Motion、Lucide React。
- 无 React Router；当前使用 `window.location.hash`。
- 无测试脚本、无自动化浏览器测试、无 Lighthouse 脚本。
- `src/App.tsx` 和 `src/index.css` 为主要实现文件。
- 数字孪生另有 `public/digital-twin/index.html` 独立实现。
- 当前源码资产约 174 MB，包含多份原图与优化图。
- 现有工作区在阶段 0 开始前已有未提交修改，重构过程中不得覆盖。

## 4. 当前阻断项

1. 首页关键数字与多项价值声明尚未核验。
2. 三大核心案例没有完整的角色、决策、协作、验证与结果字段。
3. 现有页面类型不统一，长图和 iframe 不能直接形成招聘证据链。
4. 项目路由缺少页面级元数据，不适合最终分享与 SEO。
5. 关键资产体积过大，尚未建立性能预算。
6. 部分视觉按钮无实际行为，存在可用性和可信度风险。
7. 缺少简历 PDF 与可核验奖项资料。
